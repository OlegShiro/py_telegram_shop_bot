# Python Telegram Shop Bot

Простой телеграм бот на Python 3 с реализацией функционала интернет-магазина цифровых товаров и возможностью оплаты через различные платёжные сервисы.

## Цели проекта

- Вспомнить разработку на Python;
- Вспомнить разработку на aiogram3;
- Вспомнить как работать с базами данных;
- Вспомнить как работать с Git;
- Научиться реализовывать систему оплаты через платёжные сервисы.
